Privacy Policy for GitStats Widget App

Effective Date: October 10, 2025

This Privacy Policy describes how we ("we," "us," or "our") collect, use, and disclose information in connection with your use of the GitStats Widget mobile application (the "App"). We are committed to protecting your privacy. This policy outlines our practices concerning the data we handle, including public GitHub contribution data and limited anonymous analytics data used to improve app functionality.

1. Information We Collect

The App is designed to minimize personal data collection. We only process data necessary for core functionality and app improvement.

A. Information Voluntarily Provided by You

  • GitHub Username: The public GitHub username you enter to fetch contribution data. This identifier is public and not considered sensitive.
  • Widget Configuration Data (Future Use): We may use Firebase (Firestore) in the future to store non-sensitive settings related to your widget's appearance, such as theme or layout preferences.

B. Anonymous Authentication & Firestore Usage Data

To ensure seamless app operation and manage premium features, the App uses Firebase Anonymous Authentication and Firebase Firestore. When you launch the app:

  • A unique anonymous user ID is generated (no personal info like name or email is collected).
  • We record certain app activity timestamps (e.g., when the app is opened or first installed).
  • We may store non-personal information, such as whether you have a premium subscription or related purchase identifiers (e.g., order ID).

All information stored in Firestore is anonymous, used solely for analytics and premium tracking, and is never shared or sold.

C. Information Automatically Collected (Usage Data)

We use third-party services, including Firebase Analytics and Crashlytics, to improve stability and performance.

  • Device type, OS version, and app version.
  • Anonymous session and interaction data.
  • Crash logs and technical diagnostics.

D. Third-Party Data (GitHub API)

The App fetches public data from the GitHub API based on the username you enter. We do not collect or store private repository or user information.

2. How We Use the Information

  • Core Functionality: To fetch and display your GitHub contribution graph in the widget.
  • Anonymous User Tracking: To identify anonymous users and record when the app is opened, ensuring accurate usage analytics and premium feature management.
  • Premium Management: To track whether a user has purchased or restored premium features (using anonymous identifiers and purchase order IDs).
  • App Improvement: To analyze app performance and fix crashes using Firebase Analytics and Crashlytics.
  • Personalization: To remember your last-used GitHub username and widget preferences.

3. Data Storage and Security

  • Local Storage: Your GitHub username is stored securely on your device.
  • Firebase Firestore: Used for anonymous user tracking, premium management, and timestamp logging. All stored data is anonymous.
  • Security Measures: We use encryption and secure access protocols provided by Google Firebase.

4. Disclosure of Information to Third Parties

We do not sell or share user data. Anonymous information is stored only within Firebase for essential app functionality.

5. Third-Party Service Providers

  • GitHub API: For fetching public contribution data.
  • Google Firebase: For authentication, Firestore, analytics, and crash reporting. Review policy here: Firebase Privacy Policy.

6. Children's Privacy

This app is not directed to individuals under 13 years old. We do not knowingly collect identifiable information from children.

7. Compliance with GDPR and CCPA

We respect user rights globally. Since data is stored anonymously, it cannot be used to personally identify or track you.

8. Changes to This Privacy Policy

We may update this policy from time to time and post changes within the App.
